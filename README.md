# MRGS API Client PHP

composer project akeb/mrgs_api_client_php

Composer config

```json
{
    "require": {
        "akeb/mrgs_api_client_php": "^1.0.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/AKEB/mrgs_api_client_php"
        }
    ]
}
```

or

```bash
composer require akeb/mrgs_api_client_php
```
